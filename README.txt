Projet Title : ASK AROUND
Description : est un site ou on pose une question et on reçoit les réponses des utilisateurs . On peut aussi voter les réponses (Like/Dislike)


intallation:
 Lancer le fichier install.php pour recupperer les tables SQL
   inscription
   table_ask2
   table_answer

La première page est first page sur laquelle on décide de s'inscrire ou bien de se connecter.
On est ensuite rédiger vers une page d'acceuil.
Un menu apparait composer des boutons différents :
-Home
-Ask a question : pour poser une question
-My question :  les question posées par l'utilisateur
-All question : Toute les questions posées sur le site.
-Profil : accès aux infos liées au compte
-Log out :  déconnexion du site
 Sur la page d'acceuil on trouve les questions les plus récentes.

 En dessous de chaques questions il y a deux  voir trois boutons :
 -Answer : pour repondres
 -View all answers pour voir tout les réponses
 -Edit : si la question a été posé par  l'utilisateur il peut la modifier

 Dans View all answer : 
 on retrouve la question ainsi que toutes les réponses associès classé par ordre croissant de like 
 sous chaque réponses il  y a un bouton like/dislike sauf sur la réponse posté par l'utilisateur

 ENJOY
